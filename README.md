# Conflict Mediator Mod

Мод-медиатор для разрешения конфликтов между модами Minecraft Forge 1.21+ на этапе загрузки.

## Описание

Conflict Mediator перехватывает критические ошибки во время инициализации модов и предоставляет пользователю интерфейс для выбора, какой функционал отключить, чтобы избежать краша игры.

## Возможности

- **Перехват ошибок**: Автоматическое обнаружение конфликтов при загрузке модов
- **Анализ конфликтов**: Определение модов-участников конфликта из стектрейсов
- **Отключение Mixin**: Динамическое отключение конфликтующих Mixin
- **GUI**: Интерфейс для выбора разрешения конфликтов
- **Сохранение конфигурации**: Автоматическое сохранение выбранных решений
- **API для моддеров**: Простой API для пометки отключаемых модулей

## Уровень совместимости: GOLD

Мод полностью использует API forgeModCompRFC для разрешения конфликтов и активно участвует в системе обнаружения конфликтов.

## Установка

1. Установите [forgeModCompRFC](https://github.com/neverl0se/forgeModCompRFC)
2. Поместите `conflict-mediator-1.0.0.jar` в папку `mods`
3. Запустите игру

## Использование

При обнаружении конфликта модов автоматически откроется экран с опциями разрешения:

- Отключить конкретный Mixin из одного из модов
- Отключить конфликтующий функционал в моде
- Попробовать загрузить снова
- Выйти из игры

Выбранные решения сохраняются и применяются при следующем запуске.

## API для моддеров

### Аннотация @ConflictModule

Пометьте части вашего кода как отключаемые:

```java
@ConflictModule(identifier = "new_ore", description = "New ore")
public class AdvancedAIModule {
    // Ваш код
}
```

### Событие ConflictMediationEvent

Подпишитесь на событие медиации для программного разрешения конфликтов:

```java
@SubscribeEvent
public void onConflictMediation(ConflictMediationEvent event) {
    DetectedConflict conflict = event.getConflict();
    // Ваша логика разрешения
    event.setResolution("custom_resolution");
}
```
